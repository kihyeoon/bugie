# v1.1.0 릴리즈 노트

2026-03-02

## 새 기능

### 지출자(paid_by) 기능 추가
공유 가계부에서 거래를 입력할 때 **실제로 돈을 낸 사람(지출자)**을 별도로 지정할 수 있습니다.
- 입력자와 지출자가 다른 경우를 명확하게 기록 가능
- 거래 상세에서 지출자 변경 가능

### 지출자 선택 UI
- **빠른 입력**: 지출자 선택 필드 탭 → 바텀시트에서 멤버 선택
- **거래 상세**: 지출자 항목 탭 → 바텀시트에서 변경, 선택 시 체크 애니메이션

## 개선

- 거래 상세 지출자 변경 시 낙관적 업데이트 적용 (저장 전 UI 즉시 반영)

## App Store 릴리즈 노트 (한국어)

```
공유 가계부에서 지출자를 지정할 수 있습니다.
```
